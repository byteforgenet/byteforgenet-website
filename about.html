What is Byteforgenet?

Byteforgenet is a full-featured SaaS project management tool designed with a minimal footprint. It gives teams a Kanban board, a shared notes system, a team roster with role-based permissions, and a personal dashboard — all wrapped in a clean dark UI.

We believe great tooling shouldn't require a 10MB JS bundle. Every feature in Byteforgenet is built from first principles using standard web APIs.

Technology Stack

  • Pure HTML5, CSS3, and Vanilla JavaScript — no frontend framework
  • Firebase Authentication for secure Google sign-in
  • Browser localStorage for persistent, private client-side data
  • HTML5 Drag and Drop API for the Kanban board
  • CSS custom properties for full theming support
